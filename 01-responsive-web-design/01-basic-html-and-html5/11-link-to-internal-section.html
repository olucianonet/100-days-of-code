Adicionando um link para seções internas

O elemento a pode ser utilizado também para adicionar links para uma seção interna. Com ele, deve ser utilizado o atributo href indicando o endereço do recurso.

Link para a seção 5

Verificando o fonte é possível notar que no caso de link para recurso interno, é necessário definir um id como destino e como link utilizar o label do id e uma hashtag com prefixo.
